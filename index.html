King Song Unlocker

This form allows you to generate a valid unlock code for your King Song electric unicycle. For issues and discussion see its origin thread here.

If you already know your ISN, enter it directly in the second input field. Otherwise you'll need the Bluetooth MAC address of your wheel: enter this in the first input field and it will calculate both your ISN and the unlock code for you.

If it doesn't work or you encounter an issue, post a message in the thread or open an issue on Github.

Note: any given value will be overriden if a MAC is specified!
